# Notepad – Java Swing Application

A simple and lightweight Notepad application built in Java (Swing).  
This project provides basic text-editing features with a clean and user-friendly interface.

---

## 🚀 Features
- Create, open, edit, and save text files  
- Cut, copy, paste, and select all  
- Simple and clean UI using Java Swing  
- “About” dialog  
- Custom icons included  

---

## 🛠 Technologies Used
- Java (Core Java, Swing, AWT)
- JDK 8 or above
- Works in any Java IDE (VS Code, NetBeans, IntelliJ, Eclipse)

## ▶️ How to Run
1. Install **JDK 8+**
2. Clone the repository:
   ```bash
   git clone https://github.com/Mayuri2428/Notepad.git
   ```
3. Open the project in any Java IDE
4. Run the **Notepad.java** file

This Notepad app is a beginner-friendly Java Swing project designed to understand:
